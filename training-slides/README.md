# Slide Training Team Sales — Chiến dịch Seeding Kingsmen

Bộ slide thuyết trình 5 trang do **Marketing** biên soạn để **training Team Sales**
triển khai chiến dịch seeding thương hiệu kẹo Kingsmen.

## Nội dung
1. **Tổng quan** — Trang bìa & định hướng chương trình (Marketing xây dựng, Sales thực thi)
2. **Mục tiêu** — 4 mục tiêu chính + định hướng KPI
3. **Chính sách** — Nguyên tắc bắt buộc & những điều tuyệt đối tránh
4. **Hướng dẫn** — Quy trình 5 bước thực hiện seeding
5. **Thank you** — Lời cảm ơn & thông điệp

## Nhận diện thương hiệu (theo keokingsmen.com)
- **Màu chủ đạo:** teal đậm `#0b3543` / `#114654`, cyan thương hiệu `#0a92b4` / `#0a6a80` / `#7fd4e4`
- **Font:** Montserrat (tiêu đề) + Maven Pro (nội dung)

## File
- `Kingsmen-Training-Sales-Seeding.pptx` — file trình chiếu (mở & chỉnh bằng PowerPoint/Google Slides)
- `build_slides.py` — script Python (python-pptx) tạo lại file slide
- `preview/` — ảnh xem trước 5 slide

## Tạo lại slide
```bash
pip install python-pptx
python build_slides.py
```
